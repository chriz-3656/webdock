Documentation

WebDock is a lightweight, high-performance desktop workspace that combines your favorite web applications into a single, unified environment. Built with speed and minimalism in mind, WebDock acts as a dedicated operating system for your web apps.

Features

  • Native Performance: Powered by Electron's WebContentsView, every tab runs securely and efficiently.
  • Custom Applications: Add unlimited custom websites tailored to your workflow.
  • Dynamic Icons: Automatically fetches high-resolution official favicons for a premium look.
  • Custom Themes: Switch seamlessly between Dark Mode and Light Mode instantly.
  • Privacy Focused: Isolated contexts ensure each application is securely partitioned.

Keyboard Shortcuts

  • Ctrl + Shift + I : Open Developer Tools
  • Ctrl + W : Close current tab (coming soon)
  • Ctrl + T : New tab (coming soon)

Architecture Stack

Built with Electron, TypeScript, Node.js, and Vanilla CSS/JS. No bloated frontend frameworks are used, ensuring extremely low RAM usage and lightning-fast startup times.
