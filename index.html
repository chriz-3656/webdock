About WebDock

A deep dive into how WebDock gives you your productivity back.

Minimalist Architecture

WebDock is engineered using Electron's raw `WebContentsView` API. Instead of building complex browser UI loops, WebDock directly proxies native views to your operating system window.

This bypasses standard browser chrome rendering, delivering unmatched speed and drastically reducing your machine's RAM footprint compared to keeping 50 tabs open in standard browsers.

Vanilla Frontend, Maximum Speed

Zero React. Zero Vue. Zero bloat. The WebDock interface you interact with is written in 100% vanilla HTML, CSS, and TypeScript.

By removing frontend frameworks, the UI responds instantaneously and consumes practically zero resources while running in the background.

Core Features

Everything you need to master your workflow.

Sandboxed Security

Each app runs in a fully partitioned environment. No cross-site tracking, bleeding cookies, or rogue extensions spying on you.

Unified Workspace

Combine GitHub, Figma, Spotify, Slack and more into a single elegant sidebar. Stay focused and reclaim your productivity.

Native Integration

Feels like a true desktop application with custom window controls, dynamic tab suspension, and deep OS-level integration.

Installation

Get WebDock running on your machine in seconds.

bash 
# 1. Clone the official repository
git clone https://github.com/chriz-3656/webdock.git

# 2. Navigate to the directory
cd webdock

# 3. Install dependencies
npm install

# 4. Start the application
npm start

# (Optional) Build standalone executables for your OS
npm run build && npm run dist
